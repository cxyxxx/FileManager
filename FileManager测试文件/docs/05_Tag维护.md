# 05 Tag 维护

## 移动 Tag

把 `技术/待研究` 移动到 `项目/File Manager/待研究`。

预期：

- `技术` 结构模式不再显示 `待研究`。
- `项目/File Manager` 结构模式显示 `待研究`。
- 空 Tag 移动不影响文件记录。

## 解绑文件 Tag

打开 `RAG 系统设计方案.pdf`，移除 `项目/File Manager`，保留 `技术/大模型/RAG`。

预期：

- `项目` 聚合页不再显示该文件。
- `技术` 聚合页仍显示该文件。
- 文件详情页全部 Tag 列表正确更新。

## 删除空 Tag

新建 `行政/临时测试`，不绑定任何文件，然后删除。

## 删除非空 Tag

尝试删除 `行政/报销`。

预期：系统提示该 Tag 下存在文件绑定，用户取消时数据不变，用户确认时只移除关联，不删除文件。

## 防循环

尝试把 `技术` 移动到 `技术/大模型/RAG` 下。后端应拒绝并提示不能形成循环。
